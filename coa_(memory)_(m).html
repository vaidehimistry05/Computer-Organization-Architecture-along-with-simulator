Memory
  • RAM (Random Access Memory)

    • RAM is a temporary memory used to store data and instructions while the computer is running. The data in RAM is lost when the power is turned off.

  • ROM (Read Only Memory)

    • ROM is a permanent memory used to store programs and important instructions. The data stored in ROM is not lost when the power is turned off.
