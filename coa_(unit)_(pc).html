Program Counter (PC)

  • The Program Counter (PC) is a register in the control unit that stores the address of the next instruction to be executed. It helps the computer know which instruction to fetch from memory.
  • After an instruction is fetched, the Program Counter automatically increments to point to the next instruction. This ensures that instructions are executed in the correct order. The Program Counter is essential for proper program execution and control flow.
