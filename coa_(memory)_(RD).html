Control Signals
  • RD̅ (Read)

    • The RD̅ signal is used by the microprocessor to read data from memory or an input device. When this signal is active, data is transferred from memory to the processor.

  • WR̅ (Write)

    • The WR̅ signal is used to write data into memory or an output device. When this signal is active, data is transferred from the processor to memory.

  • IO/M̅ (I/O or Memory selection)

    • The IO/M̅ signal is used to select between memory and input/output operations. When IO/M̅ = 0, it indicates a memory operation. When IO/M̅ = 1, it indicates an input/output operation.
