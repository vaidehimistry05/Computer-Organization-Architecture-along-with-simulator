# Computer-Organization-Architecture-along-with-simulator
A team-based COA project demonstrating the Control Unit of a Basic Computer and Control Memory of the 8085 Microprocessor through an interactive website. Includes a timing signal simulator to visualize instruction execution using HTML, CSS, and JavaScript.
This project is an interactive web-based implementation of key Computer Organization & Architecture concepts, including the Control Unit of a Basic Computer and the Control Memory of the 8085 Microprocessor. The platform features a timing signal simulator that dynamically visualizes T-states and decoder outputs during instruction execution, helping users understand internal CPU operations in a clear and practical way. Built using HTML, CSS, and JavaScript, the project focuses on transforming theoretical concepts into an engaging learning experience.
